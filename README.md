## 📖 Contents Overview

### Naming Conventions (Chapter 02)
The course covers multiple naming conventions used in JavaScript:

- **camelCase** - Used for variables and functions
  ```javascript
  var firstName = "navi";
  function calculateTotalPrice() {}
  ```

- **snake_case** - Used for database and API keys
  ```javascript
  var first_name = "navi";
  var email_address = "navi@example.com";
  ```

- **PascalCase** - Used for classes and constructors
  ```javascript
  class UserProfile {}
  var PersonInfo = { name: "navi" };
  ```

- **UPPER_SNAKE_CASE** - Used for constants
  ```javascript
  const API_TIMEOUT = 5000;
  const MAX_AGE = 100;
  ```
